# JavaScript Array Methods

This project demonstrates the use of JavaScript array methods:

- forEach()
- map()
- filter()
- reduce()
- some()
- every()
- find()
- findIndex()

## Files
- index.html
- forEach.js
- mapFilter.js

## How to Run
Open `index.html` in browser and check console.
